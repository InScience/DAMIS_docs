292 /* MPICH2_VERSION is the version string. MPICH2_NUMVERSION is the
293  * numeric version that can be used in numeric comparisons.
294  *
295  * MPICH2_VERSION uses the following format:
296  * Version: [MAJ].[MIN].[REV][EXT][EXT_NUMBER]
297  * Example: 1.0.7rc1 has
298  * MAJ = 1
299  * MIN = 0
300  * REV = 7
301  * EXT = rc
302  * EXT_NUMBER = 1
303  *
304  * MPICH2_NUMVERSION will convert EXT to a format number:
305  * ALPHA (a) = 0
306  * BETA (b) = 1
307  * RC (rc) = 2
308  * PATCH (p) = 3
309  * Regular releases are treated as patch 0
310  *
311  * Numeric version will have 1 digit for MAJ, 2 digits for MIN, 2
312  * digits for REV, 1 digit for EXT and 2 digits for EXT_NUMBER. So,
313  * 1.0.7rc1 will have the numeric version 10007201.
314  */
